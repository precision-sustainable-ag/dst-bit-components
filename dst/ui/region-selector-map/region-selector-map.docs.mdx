## React Component for rendering an interactive Map

A Map component made using Mapbox GL.

### List of component properties:

| Name | Type | Description |
| ---- | ---- | ----------- |
| `selectorFunc` | function | callback function to set the selected region's object as its value |
| `selectedRegion` | array | initial selected region object to highlight on the map |
| `initWidth` | string | map width in css format string |
| `initHeight` | string | map height in css format string |
| `initLon` | float | initial longitude of window center at the time of loading |
| `initLat` | float | initial latitude of window center at the time of loading |
| `initStartZoom` | float | initial zoom level at the time of loading. higher values are zoomed in and vise versa |
